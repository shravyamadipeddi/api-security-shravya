# api-security-shravya
The real-world consequences of exposing an API key on GitHub?

Unauthorized usage of your API quota.
Unexpected billing charges.
Security breaches and malicious activity.
Loss of trust and compliance violations.

Why does your company’s policy prohibit logging city names?

City names reveal personal location data.
This violates privacy laws like GDPR (EU) and HIPAA (US).
Protecting user confidentiality is a core company policy.
